# Chessboard

Basic client-side chessboard in TypeScript.

Inspired by [Sebastian Lague's video](https://www.youtube.com/watch?v=U4ogK0MIzqk).

## Build the project

Compile the TypeScript files into JavaScript:

```console
$ tsc
```
